# Overlay

覆盖层组件集合。

## 职责

- 模态对话框
- 下拉菜单
- 工具提示
- 弹出层

## 组件列表

- `Modal` - 模态对话框
- `Dropdown` - 下拉菜单
- `Tooltip` - 工具提示
- `Popover` - 弹出层
- `ConfirmDialog` - 确认对话框
- `AlertDialog` - 警告对话框

## 相关文件

- `modal.go` - 模态对话框
- `dropdown.go` - 下拉菜单
- `tooltip.go` - 工具提示
- `confirm.go` - 确认对话框
